Wagtail CMS

Thu 24 September 2015

There's a new Python CMS that just got to version 1, it's called Wagtail. I like it a lot!

Here's the quick getting started guide to know the basics. You can finish it in a few minutes. Wagtail is built on Django, but compared to the latter's tutorial, you'll be on the ground running within a few minutes. It's easy!

Here's a quick list of things about it:

Wagtail Page Model

You can do a lot with just a simple Page model and CSS.

  • model.Fields or Wagtail Fields
  • streamfield!

Different from Django

  • no views
    • use get_context
    • or use template tags
      • think of it like angular.js directives

More Batteries Included

A LOT of useful functionalities built-in.

  • built-in image and document management
  • comes with it's own customizable admin dashboard
  • form-builder and response viewer
  • multiple subsites
  • optionally define a tree structure
    • parent_page_types
    • subpage_types
  • page privacy
    • global or specific password-protected pages
  • change Wagtail admin handlers
    • search_fields
    • content_panels
    • edit handlers
  • custom content blocks
