Famo.us

Mon 24 November 2014

This month's post is about the Famo.us framework. I'm going to dedicate a lot of time to learn this javascript framework because I think it's cool in a few ways.

Famo.us lets developers create smooth complex animations in the browser using its rendering and physics engine. Animations flow as if they were native code on mobile apps, except they're in the browser. Normally, browsers would have a hard time doing this because of the DOM.

The other thing I really like about it is the integration with Angular.js. I really like Angular because of how you can make some complex things really easily, with minimal code. The learning curve for both frameworks are steep but I'm quite comfortable with Angular now so it's just a matter of time before I can be as confident with Famo.us (I hope!).

By the way, I also put up Famo.us in Manila FB and Meetup groups to bring together people who are curious or want to learn more about Famo.us. I don't think this will be a large group so that'll be better for people who want to get more involved.
